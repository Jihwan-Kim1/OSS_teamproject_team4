Features
본 프로그램은 사용자에게 즐거움을 주기 위한 게임콘텐츠를 제공한다.
사용자는 총 5개 종류의 게임중에 원하는 것을 골라서 플레이 할 수 있다.
게임 종료 후에는 게임 결과에 대한 기록(Ranking Board)을 열람할 수 있다.

사용자와 어플리케이션은 GUI를 통하여 상호작용한다.
사용자는 마우스 클릭이나 키보드 방향키 입력을 통하여 게임을 컨트롤 할 수 있다.
사용자의 닉네임은 키보드를 통하여 입력받는다. 



Usage scenario
1. <웹페이지 구성>
사용자가 웹페이지에 접속한다.
웹페이지 상단에는 게임 실행화면이 있고, 웹페이지 하단에는 게임방법이 쓰여있다.

2. <게임 선택화면>
사용자가 보게되는 게임 실행화면의 첫 화면에는 게임 선택창이 나온다.
게임의 종류는 총 5종류이다.
사용자는 원하는 게임을 선택하고, '선택완료' 버튼을 클릭한다.

3. <닉네임 입력>
게임을 선택하고 나면, 사용자에게는 닉네임 입력란이 주어진다.
키보드를 이용하여 본인이 원하는 닉네임을 입력하고, 'START'버튼을 누르면 선택했던 게임이 실행된다.

4. <게임실행>
4-1. <광클 게임>: 10초라는 제한시간내에 최대한 많이 클릭을 해야하는 게임이다.
화면 상단에는 게임 종료까지 남은 시간이 표시된다.
화면 중앙에는 사용자가 클릭해야할 버튼이 있다.
화면 하단에는 총 몇회 클릭을 하였는지가 표시된다.
사용자는 게임이 시작되고 나서 최대한 빠르게 중앙의 'CLICK HERE!' 버튼을 마우스로 연타한다.
시간이 다 되면 결과화면으로 이동한다.

4-2. <컵쌓기>: 정해진 모양대로 컵을 쌓는 게임이다.
사용자가 준비가 되면 화면의 하단의 START버튼을 클릭한다.
'START'버튼을 클릭하면 화면의 좌상단에 사용자가 만들어야 할 모양이 주어짐과 동시에 초시계가 시작된다.
사용자는 화면 우측에 주어진 컵들을 마우스 커서로 끌어와서 좌상단에 주어진 모양과 똑같이 컵을 쌓는다.
사용자가 컵을 모두 쌓고, 'FINISH'버튼을 클릭하면 결과창으로 이동한다.
 --> 정답여부?????

4-3. <니편내편>: 무작위 순서로 나타나는 캐릭터들을 분류해야하는 게임이다.
사용자는 화면 중앙에 커다란 '시작하기' 버튼을 보게 된다.
'시작하기'버튼을 클릭하면 게임이 시작된다.
화면 중앙에는 두종류의 캐릭터가 무작위 순서로 일렬로 나열되어 있다.
캐릭터 A가 나오면 왼쪽 화살표, B가 나오면 오른쪽 화살표를 클릭하여 캐릭터들을 분류한다.
분류가 틀릴때마다 틀린횟수가 카운트 된다.
모든 캐릭터들을 분류하고 나면 결과화면으로 이동한다.

4-4. <숫자 순서대로 누르기>: 무작위 순서로 배치되어있는 숫자들을 빠른 속도로, 순차적으로, 전부 클릭을 해야하는 게임이다.
사용자는 난이도 선택 화면을 보게 된다.
사용자는 난이도를 고르고 '선택완료'버튼을 누른다.
이후, 시작하기 버튼을 누르면 게임이 시작된다.
화면 상단에는 시간이 얼마나 소요되고 있는지가 표시된다.
화면 중앙에는 자기가 고른 난이도에 따라서 3X3은 1부터 9까지, 4X4는 1부터 16까지, 5X5는 1부터 25까지 숫자들이 무작위 순서로 나타난다.
사용자는 1부터 순서대로 숫자들을 클릭해 나간다. 클릭한 숫자는 사라진다.
모든 숫자를 다 클릭하면 게임이 종료되며, 결과화면으로 이동한다.

4-5. <숫자 슬라이드> 무작위로 나열된 숫자들을 정렬해야하는 게임이다.
사용자는 난이도 선택 화면을 보게 된다.
사용자는 난이도를 고르고 '선택완료'버튼을 누른다.
이후, 시작하기 버튼을 누르면 게임이 시작된다.
화면 상단에는 시간이 얼마나 소요되고 있는지가 표시된다.
화면 중앙에는 자기가 고른 난이도에 따라서 3X3은 1부터 8까지 숫자들과 빈칸 한칸,
4X4는 1부터 15까지 숫자들과 빈칸 한칸, 5X5는 1부터 24까지 숫자들과 빈칸 한칸이 무작위 순서로 나타난다.
화면 우측에는 사용자가 완성해야할 퍼즐의 모습이 나타난다.
사용자는 키보드방향키나 마우스클릭을 통하여 빈칸 주변의 숫자블록을 빈칸으로 이동시킬 수 있다.
이를 이용하여 숫자 배치를 바꾸어 나가면서 모든 숫자를 화면 우측에 나타난 모습대로, 즉 순서대로 배치하면 게임이 종료되며, 결과화면으로 이동한다.


5. <결과화면>
게임을 마치면, 각 게임의 랭킹보드가 화면 중앙에 나타난다.
화면 하단에는 이 게임을 다시 할지, 다른 게임을 고를지 선택을 할 수 있는 버튼이 있다.


Task Assignment

1. 구현
본 팀이 어플리케이션을 구현함에 있어서 분담에야할 파트들은 다음과 같다.
- 웹페이지 구현
- 게임선택화면, 닉네임 입력
- 광클게임
- 컵쌓기게임
- 니편내편게임
- 숫자순서대로누르기게임
- 숫자슬라이드게임
- 결과창

이들을 다음과 같이 분담하였다.
- 김지환: 웹페이지 구현, 니편내편게임, 숫자슬라이드게임
- 지예성: 게임선택화면, 닉네임 입력, 컵쌓기게임, 결과창
- 홍형근: 광클게임, 숫자순서대로 누르기게임

클릭을해서 다른 사람이 맞은 페이지로 넘어가는 event handler까지도 각자 맡은 부분에서 구현을 하는걸로 하였다.
추가적으로 구현중에 회의가 필요한 부분이 있거나 어려운 부분이 있으면 다른 팀원과 함께 협업하기로 하였다.


2. 문서작업
문서작업또한 각자 구현을 맞은 파트에 해당하는 부분을 맡기로 하였다.
다만, 각자 맡은 부분을 문서작업을 한 이후에 양식통일 작업은 다음과 같이 분담한다.
proposal: 지예성
보고서: 김지환
발표자료: 홍형근

3. 발표
발표는 홍형근 학우가 맡아서 하기로 하였다.
